QUICK START - LOCAL DEVELOPMENT (5 minutes)
================================================================================

1. Extract the package:
   unzip concierge-medicine-website.zip
   cd concierge-medicine-website

2. Install dependencies:
   npm install

3. Start Docker services:
   npm run docker:up

4. Run development servers:
   npm run dev

5. Access the application:
   - Frontend: http://localhost:3000
   - Backend API: http://localhost:3001
   - API Health: http://localhost:3001/health

================================================================================
AWS DEPLOYMENT - PRODUCTION (30-60 minutes)
================================================================================

PREREQUISITES:
- AWS Account with appropriate permissions
- AWS CLI installed and configured
- Docker installed
- All API keys obtained:
  * Stripe (payment processing)
  * Twilio (SMS notifications)
  * SendGrid (email notifications)
  * Agora (video conferencing)
  * AWS credentials (S3, RDS, ElastiCache)

DEPLOYMENT STEPS:

1. Read the documentation:
   - Open: AWS_DEPLOYMENT_GUIDE.md
   - Review: DEPLOYMENT_CHECKLIST.md

2. Prepare AWS resources:
   - Create S3 bucket for medical records
   - Create RDS PostgreSQL database
   - Create ElastiCache Redis cluster
   - Create ECR repository
   - Set up VPC and security groups
   - Store secrets in AWS Secrets Manager

3. Configure environment:
   - Copy: .env.production to .env
   - Fill in all required values
   - Ensure all API keys are present

4. Deploy application:
   export AWS_ACCOUNT_ID=your_account_id
   export AWS_REGION=us-east-1
   chmod +x deploy.sh
   ./deploy.sh

5. Verify deployment:
   - Check ECS service status
   - Verify ALB health checks
   - Test API endpoints
   - Monitor CloudWatch logs

================================================================================
ENVIRONMENT VARIABLES
================================================================================

Backend (.env):
- DATABASE_URL - PostgreSQL connection string
- REDIS_URL - Redis connection string
- JWT_SECRET - JWT signing secret (min 32 chars)
- STRIPE_SECRET_KEY - Stripe API key
- TWILIO_ACCOUNT_SID - Twilio account ID
- SENDGRID_API_KEY - SendGrid API key
- AWS_ACCESS_KEY_ID - AWS credentials
- AWS_SECRET_ACCESS_KEY - AWS credentials
- AGORA_APP_ID - Agora application ID
- ENCRYPTION_KEY - AES-256 encryption key (64 hex chars)

Frontend (.env):
- VITE_API_URL - Backend API URL
- VITE_SOCKET_URL - WebSocket URL

TROUBLESHOOTING
================================================================================

Port Already in Use:
lsof -i :3001
kill -9 <PID>

Database Connection Error:
docker logs concierge-postgres

Docker Services Won't Start:
npm run docker:down
npm run docker:up

Module Not Found:
rm -rf node_modules
npm install
