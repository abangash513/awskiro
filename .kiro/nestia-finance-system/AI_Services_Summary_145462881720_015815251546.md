## Cost Monitoring Setup

### CloudWatch Alarms
```bash
# Create budget alert
aws budgets create-budget \
  --account-id <account-id> \
  --budget file://budget.json

# Create CloudWatch alarm for Bedrock costs
aws cloudwatch put-metric-alarm \
  --alarm-name bedrock-cost-alert \
  --alarm-description "Alert when Bedrock costs exceed $100" \
  --metric-name EstimatedCharges \
  --namespace AWS/Billing \
  --statistic Maximum \
  --period 86400 \
  --evaluation-periods 1 \
  --threshold 100 \
  --comparison-operator GreaterThanThreshold
```

## Quick Start Commands

### Check Bedrock Model Access
```bash
aws bedrock list-foundation-models --region us-east-1
```

### Create Simple Bedrock Agent
```bash
aws bedrock-agent create-agent \
  --agent-name my-first-agent \
  --foundation-model amazon.nova-2-lite-v1:0 \
  --instruction "You are a helpful assistant" \
  --region us-east-1
```

### Create Knowledge Base
```bash
aws bedrock-agent create-knowledge-base \
  --name my-knowledge-base \
  --description "Company documentation" \
  --role-arn <your-role-arn> \
  --region us-east-1
```

### Invoke Bedrock Model
```bash
aws bedrock-runtime invoke-model \
  --model-id amazon.nova-2-lite-v1:0 \
  --region us-east-1 \
  --body '{"messages":[{"role":"user","content":[{"text":"Hello!"}]}],"inferenceConfig":{"maxTokens":100}}' \
  output.json
```
